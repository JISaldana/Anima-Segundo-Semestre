¿Qué problema solucionan las props?
Permiten pasar información de un componente padre a un componente hijo, haciendo que los componentes sean dinámicos y reutilizables.

¿Qué pasa si no mando una prop?
Por defecto, la variable dentro del componente hijo será undefined
Esto puede generar errores puntuales pero para esto se usan valores por defecto para evitar que el componente falle.
